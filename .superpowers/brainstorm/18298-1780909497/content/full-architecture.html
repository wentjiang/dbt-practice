📦 Data Sources (2 sources)

Source 1 — CSV Files

property_raw.csv

Combined file containing both:
• Transaction events: transaction_id, property_id, sale_price, sale_date, buyer_id, seller_id, agent_id, sale_type, settlement_date, days_on_market
• Property attributes: property_id, address, suburb, postcode, state, bedrooms, bathrooms, land_size, property_type, year_built, listing_price

Generated by Python/Faker · ~10k transactions · ~2k properties
Source 2 — PostgreSQL schema: src_crm

owners + agents tables

• owners: owner_id, full_name, email, phone, suburb, state, registration_date, investor_flag
• agents: agent_id, full_name, email, agency_name, license_no, region, accreditation_tier, active_from, active_to

Seeded by Python/Faker · ~500 owners · ~100 agents
